## Overview

This is a solution to the Frontend Mentor Article Preview Component Challenge.
I built a responsive article preview card using semantic HTML, modern CSS (with rem and % units), and a touch of JavaScript for interactivity.

## Features

- Fully responsive design (mobile-first, adapts to desktop)
- Semantic HTML for accessibility and structure
- Flexbox and custom properties for layout
- Share button with interactive popup
- Custom styling based on the provided design images

## My Approach

- Started with semantic HTML structure for clarity and accessibility
- Used rem and % units for all sizing (no px values)
- Styled the card to match the design as closely as possible
- Made the layout responsive using media queries
- Added interactivity for the share button using JavaScript

## What I Learned

- How to structure a component with semantic HTML
- Responsive design techniques using flexible units
- How to match a design from static images
- Using Flexbox for layout and alignment
- Adding simple interactivity with JavaScript
